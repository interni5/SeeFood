# SeeFood - Умный дневник питания с ИИ

![SeeFood](https://img.shields.io/badge/SeeFood-green)
![Python](https://img.shields.io/badge/Python-3.8+-blue)
![Flask](https://img.shields.io/badge/Flask-2.3+-lightgrey)
![JavaScript](https://img.shields.io/badge/JavaScript-ES6+-yellow)

Умное приложение для отслеживания питания с использованием искусственного интеллекта для анализа блюд и предоставления персональных рекомендаций.

##  Возможности

###  Основные функции
- **Анализ блюд через ИИ** - автоматический расчет калорий и БЖУ по описанию
- **Дневник питания** - отслеживание приемов пищи по категориям (завтрак, обед, ужин, перекус)
- **Статистика и графики** - визуализация прогресса за неделю
- **Отслеживание веса** - график динамики веса и достижения
- **Персональные цели** - настройка целей по калориям и макронутриентам

###  ИИ-функции
- **Анализ блюд** - ИИ определяет калорийность и состав по текстовому описанию
- **Персональные советы** - рекомендации по питанию на основе дневного рациона
- **Автоматические цели** - расчет оптимальных целей на основе профиля пользователя

### Интерфейс
- **Адаптивный дизайн** - оптимизирован для мобильных устройств
- **Интуитивный онбординг** - пошаговое знакомство с приложением
- **Визуальная статистика** - кольца прогресса, графики и диаграммы
- **Оффлайн-работа** - данные сохраняются локально

##  Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd SeeFood
```

### 2. Установка зависимостей
```bash
# Создание виртуального окружения
python -m venv .venv

# Активация (Windows)
.venv\Scripts\activate

# Активация (macOS/Linux)
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements_new.txt
```

### 3. Настройка API ключа
Получите API ключ на [OpenRouter](https://openrouter.ai/) и добавьте его в `app_new.py`:
```python
OPENROUTER_API_KEY = "ваш_api_ключ_здесь"
```

### 4. Запуск приложения
```bash
# Запуск нового backend
python app_new.py

# Или запуск оригинального backend
python app.py
```

Приложение будет доступно по адресу: http://127.0.0.1:5000

##  Использование

### 1. Онбординг
- При первом запуске пройдите пошаговый онбординг
- Укажите ваши цели, рост, вес и уровень активности
- Приложение автоматически рассчитает оптимальные цели

### 2. Добавление блюд
- Перейдите на вкладку "Питание"
- Введите описание блюда (например: "гречка 200г, курица 150г")
- Выберите тип приема пищи
- Нажмите "Добавить" - ИИ проанализирует блюдо

### 3. Отслеживание прогресса
- На главной вкладке смотрите статистику за день
- График показывает потребление за последние 7 дней
- Кольцо прогресса отображает выполнение дневной нормы калорий

### 4. Управление весом
- На вкладке "Вес" добавляйте текущий вес
- График показывает динамику изменений
- Отслеживайте прогресс к цели

##  API эндпоинты

### Основные эндпоинты
- `GET /` - Главная страница
- `GET /api/health` - Проверка работоспособности
- `POST /api/analyze` - Анализ блюда через ИИ
- `GET /api/diary` - Получение дневника питания
- `POST /api/weight` - Добавление веса
- `GET /api/goals` - Получение целей
- `POST /api/goals` - Сохранение целей
- `POST /api/tip` - Получение совета от ИИ
- `GET /api/stats` - Получение статистики
